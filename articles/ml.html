Machine Learning

Machine learning is a subset of artificial intelligence that involves training computers to learn from data without being explicitly programmed.

Concepts in ML
  • Feature Engineering
    • Change input data form, suitable for machine learning algoritms.
  • Module Evaluation
    • Testing the performance of machine learning algoritms when testing (validation) data is inputted.
  • Overfitting
    • When training data is not filtered.Model "memorize" the data rather than "learning" from data.
  • Underfitting
    • Existing traning data is less because algorithm did not find any pattern.

Basics step to learn Deep Learning

Ability of a machine to mimic human behavior through programming is called Machine Learning.
Ability of machine to minic human behavior without explicit programming is know as Artificial Intelligence.
Ability of program to extract features, pattern from data is known as Deep Learning

Deep learning

Deep learning consist of input data, weights, bias value, activation function and output data

  • Input Data
    • Input data (here x1, x2, x3) refers to features, characteristics for example : age, salary, tenure
  • Weights
    • Weights (here w1, w2, w3,..., w9) are arbitary values at first choosen randomly and then adjusted using gradient descent.
  • Activation Function
    • Activation function are used to introduce non-linearity in the program because dataset are mostly non-linear. For example : ReLu, Sigmoid function
  • Output Data
    • Bias value are used to shift the non-linearity.
