  • Destructuring
    ARRAY DESTRUCTURING
    let arr = ['Tanmay', 'Mishra']

console.log(arr[0]) // Tanmay
console.log(arr[1]) // Mishra

**Array Destructuring**
let[first, middle, last] = arr

console.log(first) // Tanmay
console.log(middle) //undefined
console.log(last)  // Mishra
    OBJECT DESTRUCTURING
    let myObj = {
    fname: 'Tanmay',
    lname: 'Mishra'
}

console.log(myObj.fname) // Tanmay
console.log(myObj.lname) // Mishra

**Object Destructuring**
let {fname, lname} = myObj

console.log(fname) // Tanmay
console.log(lname) // Mishra
