HTML5 APIs

13 May 2023 by Tanmay

There are basically three types of API in HTML5 :

  • Web Storage API
    The Web Storage API provides a way to store data in the user's browser. It offers two mechanisms: localStorage and sessionStorage. Both provide a key-value pair storage interface
  • Web Workers API:
    The Web Workers API allows web applications to run JavaScript code in the background, separate from the main UI thread.
  • Geolocation API:
    The Geolocation API allows web applications to access the user's geographical location information.

Display - Flexbox

08 May 2023 by Tanmay

Flexbox is layout module have following terms

Parent Container
  • Justify Content

    Horizontal, x-axis, Landscape, main-axis

  • Align Items (space b/w rows)

    Vertical, y-axis, Portrait, cross-axis

  • Align Content (no space b/w rows)

    space-around, space-between, start, end, stretch

  • Flex-Wrap

    nowrap, wrap, wrap-reverse

Child Container
  • Align Self (align child in y-axis)

    flex-start, flex-end, center, stretch

  • Order
  • Flex-basis

    Control width of child container individually

  • Flex-grow

    Higher the number higher it will grow relative to other children

  • Flex-shrink

    Higher the number higher it will shrink relative to other children

1.0 Flex-direction:row (default)
1.1 Flex-direction:row-reverse
2.0 Flex-direction:column
2.1 Flex-direction:column-reverse
