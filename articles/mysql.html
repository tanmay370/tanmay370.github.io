MySql

06 September 2023 by Tanmay

Database Commands

  1. Create database
    create database student;
    Create database databaseName;
  2. Show databases
    show databases;
    To see all databases in MySql
  3. Switch databases
    use databaseName;
    To go into particular database in MySql

Tables Commands

  1. Show Tables
    show tables;
    To see tables in particular database
  2. Create Table
    create table table_name (
    Attribute1 datatype constraints,
    Attribute2 datatype,
    primary key (Attribute1));
  3. Table Description
    describe table_name;
    To see schema of table
  4. Table Data Description
    select * from table_name;
    To see full table with data
  5. Change Table name
    alter table faculty rename staff;
    alter table tableName rename aliasname

Rows & Columns Commands

  1. Insert Data
    insert into table_name values ( 1, "Ashish");
    To enter values in tables
  2. Update Column value
    update student set age = 23 where name = 'Sandhya';
    update table_name set attributeName = aliasname where attribute = existing value;
  3. Change table schema
    describe table_name;
    alter table table_name modify attribute datatype;
  4. Add column
    alter table student add Fees double;
    alter table table_name add attribute datatype

CRUD Commands

  1. Count number of rows
    select count(*) from tableName
  2. Reading data from Two tables
    select attribute_name, attribute_name from
    table_name, table_name
    where
    tablename.attribute = tablename.attribute
  3. Reading data from two tables using aliasname for table
    select attribute_name, attribute_name from
    table_name whitespace aliasname,
    table_name whitespace aliasname
    where
    aliasname.attribute = aliasname.attribute
  4. Reading data from two tables using aliasname for table
    select attribute_name, attribute_name from
    table_name whitespace aliasname,
    table_name whitespace aliasname
    where
    aliasname.attribute = aliasname.attribute
