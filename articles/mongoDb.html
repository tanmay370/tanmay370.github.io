MongoDb

06 September 2023 by Tanmay

mongo: The command -line shell that connects to a specific instance of mongod.

mongod: It is basically the host process for the [Mongo Daemon] database.

Terminology bt MySql & MongoDb

SQL term MongoDb term
Database Database
Tables collections
Rows document (BSON)
Columns fields

MondoDb Compass: Desktop version of mongoDb.It gives cloud facilities

Database Commands

  1. View all databases
    show dbs
  2. Create a new or switch databases
    use dbName
  3. View current database
    db
  4. Delete database
    db.dropDatabase

Collection Commands

  1. Show collections
    show collections
  2. Create a collection named 'comments'
    db.createCollection ('comments')
  3. Drop a collection named 'comments'
    db.comments.drop()

Document(Row) Commands

  1. Show all rows in a collection
    db.comments.find()
  2. Show all rows in a collection in pretty format
    db.comments.find().pretty()
  3. Insert one Row
    db.comments.insert({ }) --> Takes an object
  4. Insert many Rows
    db.comments.insertMany([ { } ]) --> Takes an array
  5. Find the first row matching the object
    db.comments.findOne ({name: 'Sara'})

Search in Mongodb

  1. Search in MongoDb database
    db.comments.find( { lang: 'Python' } )
  2. Limit the number of rows in output
    db.comments.find().limit(2)
  3. Count the number of rows in output
    db.comments.find().count()
  4. Count the number of rows according to a filter
    db.comments.find( { lang: 'Python' } ).count()

Sort in MongoDb database

  1. Sort in increasing order
    db.comments.find().sort ( { member_since : 1 } ).pretty()
  2. Sort in decreasing order
    db.comments.find().sort ( { member_since : 1 } ).pretty()
