29 August 2023 by Tanmay

Initializing a project with git

  • git status

    On branch master --> Branch named master
    nothing to commit , working tree clean

  • git init

    init command initialise a .git repository

  • Add File

    File is added to the staging area. Shortcut way to add all available files to the staging area is to run command 

     git add .

  • Commit (Taking screenshot)

    Commiting (taking screenshot) of code exist that time when we run the command below here -m flag stands for message and "Initial commit" stands for comment user defined.

     git commit -m "Initial commit"

  • Delete or Unmake a repo

    Delete a file or folder
    .git is like a container contained compressed snapshot of previous version

     git rm filename
    From working tree and staging area
     git rm --cached filename
